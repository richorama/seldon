# @seldon/llm

Thin LLM **provider abstraction** for Seldon. Azure OpenAI first (v1 API,
reasoning-model aware), plus a deterministic mock provider for tests.

Part of [Seldon](https://github.com/richorama/seldon).

## Install

```bash
npm install @seldon/llm
```

See the [repository](https://github.com/richorama/seldon) for docs and examples.
