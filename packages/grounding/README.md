# @seldon/grounding

Web-grounding for Seldon: a `Fetcher` interface (with a Wikipedia REST
implementation) and a persistent JSON + Markdown **fact cache** with TTL.

Part of [Seldon](https://github.com/richorama/seldon).

## Install

```bash
npm install @seldon/grounding
```

See the [repository](https://github.com/richorama/seldon) for docs and examples.
