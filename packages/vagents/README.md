# @seldon/vagents

Orleans-style **virtual agents** (vagents) for LLM deliberations: on-demand
activation, a barrier-synchronised turn loop, bounded concurrency and hard caps.

A generic, domain-agnostic runtime — the Seldon prediction engine is one host
built on top of it. Part of [Seldon](https://github.com/richorama/seldon).

## Install

```bash
npm install @seldon/vagents
```

See the [repository](https://github.com/richorama/seldon) for docs and examples.
