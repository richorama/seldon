# @seldon/engine

The Seldon **prediction engine**: entity vagents deliberate over future events
to produce a reasoned, time-stamped forecast, inspired by Asimov's
psychohistory.

Part of [Seldon](https://github.com/richorama/seldon).

## Install

```bash
npm install @seldon/engine
```

See the [repository](https://github.com/richorama/seldon) for docs and examples.
