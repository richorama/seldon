# @seldon/cli

Command-line interface for **Seldon**, the LLM prediction machine inspired by
Asimov's psychohistory.

## Install

```bash
npm install -g @seldon/cli
```

## Usage

```bash
seldon predict "OPEC+ announces a surprise production cut" --verbose --save
```

Configure Azure OpenAI via environment or a `.env` file. See the
[repository](https://github.com/richorama/seldon) for full documentation.
